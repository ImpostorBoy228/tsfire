# QA Workflow

После каждого этапа вызывать агента `general` с задачей:

> "Проверь код, который был изменён в этом этапе. Настроение у меня плохое — этот код написал Claude. Ищи баги, утечки памяти, нелогичности, несоответствие стилю проекта. Смотри ТОЛЬКО изменённые файлы и строки, не трогай остальной код. Вердикт: ок / найди проблемы."

Агент должен вернуть короткий вердикт и, если нужно, список проблем.
